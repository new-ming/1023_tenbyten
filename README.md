## 개요 
* 프로젝트 목표 : `JSP` 와 `Servlet`을 사용한 프로젝트
<br>

## 기술 스택
  - Language : `Java(11)`, `JavaScript(ES6)`
  - Library & Framework : `Servlet(4.0)`
  - Database : `MySql(8.0.34.0)`
  - Target : `Web Browser`
  - Tool : `Eclipse(4.29.0)`
  - ETC : `Git`

## 핵심 기능
#### 회원정보
  - 로그인/로그아웃
  - 회원가입
  - 개인정보 수정
  - 회원 탈퇴

#### 문의
  - 문의 등록
  - 문의 수정
  - 문의 삭제
